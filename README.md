# XmindToExcelJava

Xmind思维导图转Excel格式的测试用例

支持上传指定格式的Xmind文件，转换为测试用例

Excel中包含以下项目：

NO，项目，模块，功能，子功能，操作步骤，预期结果，实际结果

注意：编写Xmind用例时，需要在步骤和预期结果中插入一个“预期结果”子主题，作为占位符使用
